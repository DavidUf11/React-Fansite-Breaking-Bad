## Felina

A Wyncode midterm project, Felina is a web app that allows fans of the AMC TV series _Breaking Bad_ explore their favorite characters and test their knowledge of the show.


## Technologies Used
- React
- React Router Dom
- Node.js
- Express
- Bootstrap
- Sweetalert
